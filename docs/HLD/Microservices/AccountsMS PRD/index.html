Here’s the complete project requirement specification for the Accounts Microservice, implemented using Spring Boot (non-reactive Spring MVC), with suggestions for code styling enforcement using Checkstyle and other good practices:

📘 Accounts Microservice – Project Specification​

📌 Purpose​

To manage various types of accounts like:

  • Savings / Current Bank Accounts
  • Credit Card Accounts
  • Loan Accounts
  • Investment Accounts (e.g., stocks, bonds)
  • Digital Wallets

🔧 Tech Stack​

LayerTechnology
LanguageJava 17+
FrameworkSpring Boot (Spring MVC)
DatabasePostgreSQL
MessagingApache Kafka
CacheSpring Cache + Redis
API SecurityJWT + HTTPS
ConfigurationSpring Cloud Config
Secret ManagementHashiCorp Vault
ObservabilityPrometheus + Grafana + Loki
API DocumentationSpringDoc OpenAPI (Swagger)
Code StyleCheckstyle + Google Java Format
Linting/Static AnalysisPMD + SpotBugs
ContainerizationDocker
OrchestrationKubernetes
Unit TestingJUnit 5 + Mockito + Testcontainers

🔐 Security & Access​

  • JWT for authentication
  • Role-based access control (e.g., USER, ADMIN)
  • HTTPS via TLS termination at ingress (e.g., NGINX)

🧠 Responsibilities​

  • Create/update/soft-delete accounts
  • Store detailed financial info depending on account type
  • Fetch accounts by user
  • Map accounts to user via another microservice
  • Send Kafka events: ACCOUNT_CREATED, ACCOUNT_UPDATED, ACCOUNT_CLOSED

🔄 Inter-Service Communication​

Target MicroserviceProtocolPurpose
User MicroserviceRESTValidate user existence
Budget MicroserviceKafkaNotify budgets of new/closed accounts
Expense MicroserviceRESTAccount validation for expense creation

ColumnTypeDescription
account_numberUUID (PK)Unique identifier for account
account_nameVARCHAR(20)Name of the account
institution_nameVARCHAR(50)Bank/Institution name
account_type_idINTFK to account_type
account_statusENUMOPEN, CLOSED, OPENING, CLOSING, LOCKED
currencyVARCHAR(10)Currency type (e.g., USD, INR)
created_atTIMESTAMPCreation time
updated_atTIMESTAMPLast update time
descriptionTEXTOptional details

ColumnTypeDescription
account_type_idINT (PK)Auto-increment ID

ColumnTypeNotes

interest_rateDECIMALFor savings/loan/investment

loan_amountDECIMALLoan only

🔁 Caching​

  • Redis + Spring Cache

    • Cache account types and institution info
    • TTL: 24 hours, invalidated on update via cache eviction
